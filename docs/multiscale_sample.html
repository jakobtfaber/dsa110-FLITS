casey — zero resolved diffractive scales

For casey the multiscale analysis resolves zero diffractive scintillation scales. The DSA ~5–6 MHz feature (m²~0.5, BIC N=1) is NOT scintillation but a non-stationary intrinsic/instrumental bandpass envelope, partly inflated by edge channels where the DSA gain crosses zero: it is annihilated by a cubic detrend (6→0.29 MHz), present only in the lower half-band and absent in the upper (a stationary screen cannot vanish across half a band), and shrinks to ~3.5 MHz with m² halved when low-gain channels are masked — whereas an injected true scintle survives all three tests. Cross-band width ratio implies α~0.9 (vs ~4 for diffraction); the host τ-screen Δνd (~0.001–0.007 MHz) is ~1000× below any measured width and unresolvable. No unresolved host component (the fine-channel m² rise is not significant once N-weighted, t=0.54) and no 2nd screen (BIC strongly favors one component). Conservatively: strong intrinsic spectral structure, no resolved scintillation scale.

wilhelm — one resolved non-diffractive scale + a non-host unresolved component

Wilhelm shows one robustly resolved DSA scale, ~2.4–2.7 MHz (lag-1 ACF 6.5–7σ above the noise floor; it survives only when the true channel ordering is preserved — a channel shuffle collapses it to ~0.3–0.6 MHz, so the correlation is real and lives in frequency). BIC decisively prefers a single Lorentzian (0/400 bootstraps favor N=2; the 2nd component contributes 0.000% RSS). But this width is NOT diffractive scintillation: splitting the DSA band gives a strongly frequency-dependent width (0.95→2.62 MHz; implied within-band α~11–15, unphysical) with no plateau across the ladder — most consistent with a fixed/partially-unresolved spectral envelope. A separate real unresolved component IS detected (noise-corrected m² rises 0.012→0.047, t=5.5) — but its forward Δνd ~0.16–0.66 MHz is ~100–430× the host τ-screen prediction (~0.0015 MHz), so it is a real sub-channel scale that is NOT the host screen. CHIME is S/N-marginal, so the cross-band ν-test is corroborating-only; all conclusions rest on DSA-internal evidence. Net: one resolved (non-diffractive) scale + one real non-host unresolved component; no 2nd screen, no host scintle.
