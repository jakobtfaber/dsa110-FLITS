# Chat brainstorm — short take (pre-report)

From Undermind chat in project **CHIME-DSA FRB Open Questions**, before the 50-paper deep search report.

## Bottom-line question

When α is **measured** (not assumed) and **DM + scattering are modeled jointly** on dual-band localized FRBs, do **any** require **intervening-halo/CGM scattering**, or is scattering mostly **host-dominated** (± Milky Way)?

## Ranked gaps (chat, not the full report table)

1. **Strongest gap:** measured α plus screen attribution on the **same** localized bursts (prior work usually has scattering *or* foreground DM context, not both).
2. **Second:** falsification test of “halos add DM but not scattering” per burst.
3. **Methods:** component-aware dual-band fitting — unmodeled substructure can bias α.

## Already partly done (not flagship)

- Screen localization method (CRAFT: 3/10 two-screen).
- Halo DM excess in large samples.
- Halo **scattering** likely rare vs halo **DM**.
- Profile complexity contaminates scattering fits.

## Not good flagship for N=12

Population halo-scattering incidence, CGM profile / baryon fraction, mNFW discrimination, morphology stats — need hundreds–thousands of FRBs.

## Suggested paper order (chat)

1. Main: burst-by-burst screen attribution (dual-band τ+α, DM budgets, foreground halos).
2. Methods: profile-subcomponent bias in α.
3. Optional: best Δν cases as explicit two-screen scintillometry.
